【概要】

・使用言語：PHP、MySQL、JavaScript、HTML、CSS

・フレームワーク：JQuery

・プログラミング学習期間約２ヶ月で作成開始。作成時間８０時間。

・基本的な言語学習と理解の為、WEB サービスの基本機能、バリデーションチェックからページング機能など各種機能、DB(6 テーブル）の例外処理による登録などは全て PHP と MySQL を使い機能実装しています。

【サービス概要】

・ビジネス書の専門サイトです。

・アカウント作成し、自分の好きな本を登録できます。

・その際、ジャンルや対象者を指定できます。

・検索機能を用い、利用者に合わせ容易に調べられるようにしました。

・また本の詳細ページにてコメントの投稿を行うことが出来ます。

・コメント時には年齢が表示されるので、別の利用者は購入の参考に出来ます。

・そして詳細ページには予め、本の購入先へとリンクする URL を登録しているので、購入ボタンから購入ページに画面遷移する仕組みを作っています。

・その他、アカウント作成・ログイン（ログイン保持機能有り）・ログアウト・アカウント削除・Email リマインダーの基本機能。

・またDB の設計・例外処理を利用した登録まで全て、フレームワークは使わずPHP と MySQL を用いコードを書いています。
